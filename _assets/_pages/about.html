About

This webpage aims to have all Versions and all ROM Hacks of Super Mario 64 available. This webproject as a whole is maintained by Tomatobird8 and MarvJungs If you like what you see and would like to see possible improvements and further maintance of this website, feel free to put out a donation
Technical Details: The Website is hosted by a domain provider located in Germany and the content is generated using PHP with HTMP DOM Manipulation with JavaScript and Data Storage in CSV- and JSON-Files.

Privacy Notice

Upon visiting this webpage, we will collect the following information stored in a log-file but for no more than 60 days:
  • IP Address
  • Timestamp
  • HTTP Request
  • HTTP Response Code
  • Accessed Resource
  • Browser and Operating System
This data may be used to do Data Analysis in the near future but as of the date writing this (06th May 2023), no data processing is done. This section will be updated once needed.
